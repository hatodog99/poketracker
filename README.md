# PokéTracker

## Requirements
- XAMPP
- PHP
- MySQL

## Setup

1. Clone repository
2. Put folder inside C:/xampp/htdocs/
3. Start Apache and MySQL in XAMPP
4. Open phpMyAdmin in browser by pasting this in the search bar: http://localhost/phpmyadmin
5. Create database in phpMyAdmin:
   poketracker_db
6. Import:
   database/poketracker_db.sql
7. Visit:
   http://localhost/poketracker
